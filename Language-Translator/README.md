# 🌍 AI Language Translator Tool (Python Tkinter Desktop App)

A powerful desktop-based Language Translator Application built using Python.  
It provides real-time translation, optional text-to-speech, and translation history tracking using a simple and interactive Tkinter GUI.

# 🚀 Features

- 🌐 Translate text between 100+ languages using Google Translate API  
- 🧠 Auto language detection support  
- 🔊 Text-to-Speech (speech output for supported languages)  
- 📜 Translation history tracking  
- 🖥️ Clean and responsive Tkinter GUI  
- ⚡ Fast and lightweight performance  
- 🔄 Easy language switching via dropdown  

# 🛠️ Tech Stack

- Python 3.x  
- Tkinter (GUI Framework)  
- googletrans (Translation API)  
- gTTS (Text-to-Speech)  
- pygame (Audio playback)  
- unidecode (Text normalization)  
- datetime (Timestamp tracking)  
- threading (Smooth performance)

# ⚙️ Installation & Setup

Step 1: Clone Repository  
git clone https://github.com/Shravya886/SAM_AI_Tech_Tasks/edit/main/Language-Translator 

Step 2: Go to Project Folder  
cd language-translator  

Step 3: Install Dependencies  
pip install googletrans==4.0.0-rc1 gtts pygame unidecode  

Step 4: Run Application  
python main.py  

# 🖥️ How to Use

1. Open the app  
2. Enter text in input box  
3. Select language from dropdown  
4. Click Translate  
5. View translated output instantly  
6. Click Speak to hear translation  
7. Check translation history  

# 📜 History Feature

Stores:
- Input text  
- Translated text  
- Timestamp of translation  

# 🔊 Text-to-Speech (TTS)

Uses gTTS (Google Text-to-Speech) to convert translated text into audio and plays it using pygame.
